# MyGPU: GPU 관리 유틸리티

MyGPU는 NVIDIA `nvidia-smi`의 경량 버전으로, 우아한 웹 대시보드를 갖춘 컴팩트한 GPU 관리 도구입니다.

## 기능

- **실시간 모니터링**: GPU 및 시스템 메트릭(사용률, VRAM, 전력, 온도)을 제공합니다.
- **관리 및 강제 실행**: VRAM 제한 설정, 프로세스 자동 종료(관리자 전용), 모니터링 목록을 통한 특정 PID 또는 프로세스 이름 모니터링.
- **벤치마킹 및 시뮬레이션**: 열 스로틀링 및 안정성 테스트를 위한 GEMM 워크로드, 3D 입자 물리 시뮬레이션을 통한 GPU 부하 시각화 등.

## 로드맵 및 미래 작업

기여를 환영합니다! 미래에 다룰 주요 사항은 다음과 같습니다.

- **다중 GPU 지원**: 다중 카드 설정 및 NVLink 토폴로지에 대한 향상된 처리.
- **컨테이너화**: Docker 공식 지원을 통한 컨테이너 환경에서의 간편 배포.
- **원격 액세스**: SSH 터널링 통합 및 보안 원격 관리.
- **플랫폼 간 지원**:
  - [ ] Linux 지원(Ubuntu/Debian 중점).
  - [ ] Apple Silicon을 위한 macOS 지원.
- **하드웨어 독립성**:
  - [ ] AMD ROCm 지원.
  - [ ] Intel Arc 지원.
- **다국어 문서화**: GitHub에서 가장 인기 있는 언어로 지원.

## 요구 사항

- **OS**: Windows 10/11
- **Python**: 3.10 이상
- **하드웨어**: NVIDIA GPU 및 설치 드라이버
- **CUDA**: 12.x(벤치마킹/시뮬레이션 기능 사용 시 엄격히 필요)

## 설치

도구는 모듈식 설치를 지원하여 여러 요구 사항에 맞게 설치할 수 있습니다.

### 1. 최소(CLI 전용)

헤드리스 서버 또는 백그라운드 모니터링에 적합합니다.

- 명령줄 인터페이스
- 기본 시스템/GPU 메트릭

### 2. 표준(CLI + 웹 UI)

대부분의 사용자에게 적합합니다.

- 웹 대시보드 포함
- REST API 엔드포인트
- 실시간 차트
- 하지만 시뮬레이션 또는 벤치마킹 없음

### 3. 전체(표준 + 시각화)

개발 및 스트레스 테스트에 적합합니다.

- 시뮬레이션 포함
- PyTorch/CuPy 의존성을 통한 벤치마킹

### 빠른 시작

1. **다운로드** 최신 릴리스 또는 저장소를 클론합니다.
2. **설정 실행**:

   ```powershell
   .\setup.ps1
   ```

3. **시작**:

```powershell
# 웹 대시보드 시작(표준/전체)
python health_monitor.py web

# CLI 시작
python health_monitor.py cli
```
