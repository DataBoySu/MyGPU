# MyGPU: 軽量GPU管理ユーティリティ

*MyGPU* は、NVIDIA `nvidia-smi` のコンパクトなラッパーであり、エレガントなウェブダッシュボードを備えたツールです。

## 特徴

- **リアルタイム監視**: GPUとシステムの詳細なメトリクス（利用率、VRAM、電力、温度など）。
- **管理と強制実行**: VRAMキャップ（GPUあたりのVRAM使用量制限）、自動終了（VRAMポリシー違反時のプロセス終了、管理者のみ）、ウォッチリスト（特定のPIDまたはプロセス名を監視）。
- **ベンチマークとシミュレーション**: 熱低減と安定性をテストするためのGEMMワークロード、3D粒子物理学シミュレーションによるGPU負荷の可視化。

## 要件

- **OS**: Windows 10/11
- **Python**: 3.10以上
- **ハードウェア**: NVIDIA GPUとインストールされたドライバー
- **CUDA**: 12.xツールキット（ベンチマーク/シミュレーション機能を有効にするために厳密に必要）。
- **ネットワーク**: インターネット接続（WebダッシュボードとAPIエンドポイントのため）

## インストール

ツールには、ニーズに合わせてモジュールインストールが可能です。

### 1. 最小（CLIのみ）

ヘッドレスサーバーやバックグラウンド監視に最適です。

- コマンドラインツール
- 基本的なシステム/GPUメトリクス

### 2. 標準（CLI + Web UI）

ほとんどのユーザーに最適です。

- Webダッシュボードを含む
- REST APIエンドポイント
- リアルタイムチャート

### 3. フル（標準 + 視覚化）

開発とストレステストに最適です。

- 粒子シミュレーションを含む
- PyTorch/CuPy依存関係によるベンチマーク

### クイックスタート

1. **ダウンロード** 最新リリースまたはリポジトリをクローンします。
2. **セットアップを実行**

   ```powershell
   .\setup.ps1
   ```

3. **起動**

   ```powershell
   # Webダッシュボードを起動（標準/フル）
   python health_monitor.py web

   # CLIを起動
   python health_monitor.py cli
   ```
